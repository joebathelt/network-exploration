Network Exploration

The interaction between elements has become a focus in psychological research in recent years. This new approach emphasises the interaction between elements like symptoms of a disorder in order to understand the synergies. This network perspective can provide many new insights into psychological phenomena. However, networks can also grow rapidly in complexity once more than a just a few elements are included. We created this visualisation to aid in the active exploration of networks. Using this web app, you can upload your own networks and explore the effect of removing connections.

Uploading your data:

  1. This web app can load networks generated in Bootnet
  2. Networks generated using other software can also be represented but need to be converted to this format
